# Syntero Knowledge Base

Obsidian vault inside the Syntero repo (`shadcn-ui/docs-obsidian`).

## Type

Navigation

## Structure

| Folder | Content |
|---|---|
| **Estimation/** | Domain artifacts, pipeline, wizard flow, engine |
| **AI/** | Endpoints, prompts, interview flow |
| **Architecture/** | Stack overview, constraints |
| **Data Model/** | Schema, migrations |
| **Plans/** | Implementation milestones |

## Entry Points

- Open [[Home]] for navigation
- Open [[Estimation Pipeline]] for the semantic flow
- Use **Graph View** to explore artifact relationships

## Usage

- Open this folder as a vault in Obsidian
- Notes use `[[wikilinks]]` for semantic linking
- The graph reflects real code relationships, not just hierarchy
